# 用户使用指南

## 功能
- 选择 **安装渠道 / 出价类型 / 国家 / APP** 四种筛选  
- 切换 **数据展示模式（原始 / 7 日移动均值）**、**Y 轴刻度（线性 / 对数）**  
- 图例可点击隐藏 / 显示曲线  
- 悬停查看精确数值  
- 红色 100% 回本线  
- **支持导入 ROI 数据 CSV 文件，自动写入数据库**

---

## 操作步骤
1. 打开首页（`/`），从下拉框选择 APP、国家、渠道、出价类型等筛选条件  
2. 在 **“数据展示模式”** 中选择 **“移动平均值”** 或 **“原始数据”**  
3. 在 **“Y 轴刻度”** 中选择 **“线性”** 或 **“对数”**  
4. 点击图例可切换曲线显隐  
5. 点击右上角导航菜单 **“导入数据”** 进入 CSV 数据导入页面  

---

## 0% 的含义
- **真实 0%**：该周期确实为 0  
- **日期不足 0%**：该天距离数据截止日不足对应周期，暂以 0 显示  

---

## CSV 数据导入

### 📍 入口  
点击顶部导航中的 **“📤 导入数据”**，或直接访问 `/import` 页面。
